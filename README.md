## About

[Karaoke Nite](https://karaokenite.co) is a platform where everyone can sing their favorite tunes and hang out with friends. It can be accessed using a VR headset or on the browser.

Currently supported on:

- Desktop
- Mobile
- Oculus Quest (VR)
- Oculus Rift (VR)
- HTC Vive (VR)
